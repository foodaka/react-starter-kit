
# React Redux Starter Kit

This repo is bootstrapped with react-create-app and has some additional technologies out of the box

* React
* Redux
* Redux Dev Tools
* Eslint
* Husky pre commit
* Prettier formatting

## Setup
Install packages

```bash
yarn install 
```

Run Server
```bash 
yarn start 
```

Check if all validations pass
```bash
yarn run validate 
```

Auto format code using prettier
```bash 
yarn run format
```
